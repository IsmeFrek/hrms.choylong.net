# របៀបប្រើប្រាស់ Import Excel សម្រាប់ប្រតិទិនការងារ

## ជំហានទី១: Export Template

1. ចូលទៅកាន់ផេក "ប្រតិទិនការងារ"
2. ជ្រើសរើស **ខែ** ដែលចង់គ្រប់គ្រង
3. ចុចប៊ូតុង **"Export Excel"** (ពណ៌ស្វាយ)
4. ឯកសារ Excel នឹងត្រូវបានទាញយក

## ជំហានទី២: កែសម្រួលទិន្នន័យក្នុង Excel

### រចនាសម្ព័ន្ធតារាង:

```
| ល.រ | ទូរស័ព្ទ | លេខកាត | ឈ្មោះ | តួនាទី | ទី១ | ទី២ | ទី៣ | ... | ទី៣១ |
```

### និមិត្តសញ្ញាប្រើប្រាស់:

- **✓** = បានបញ្ចប់ (Completed)
- **○** = បានកំណត់ (Scheduled)
- **✗** = បានលុបចោល (Cancelled)
- **ទទេ ឬ -** = មិនមានតារាងការងារ

### ឧទាហរណ៍:

```excel
ល.រ | ទូរស័ព្ទ      | លេខកាត | ឈ្មោះ        | តួនាទី    | ទី១ | ទី២ | ទី៣ | ទី៤ | ទី៥
1   | 012-345-678  | E001   | សុខ ចន្ទា   | គ្រូបង្រៀន | ✓  | ✓  | ✗  | ○  | ✓
2   | 012-987-654  | E002   | មុំ សុផល   | គ្រូបង្រៀន | ○  | ○  | ○  | ✓  | ○
```

## ជំហានទី៣: Import ទិន្នន័យ

1. ចុចប៊ូតុង **"Import Excel"** (ពណ៌ទឹកក្រូច)
2. ជ្រើសរើសឯកសារ Excel ដែលបានកែសម្រួល
3. ពិនិត្យមើលទិន្នន័យក្នុងផ្ទាំង Preview
4. ចុចប៊ូតុង **"បញ្ជាក់ Import"** ដើម្បីរក្សាទុក

## ចំណាំសំខាន់:

✅ **ត្រូវរក្សាទុក ឈ្មោះបុគ្គលិក ឬ លេខកាត ឲ្យត្រឹមត្រូវ** ដើម្បីឲ្យប្រព័ន្ធអាចបង្កើតការតភ្ជាប់បាន

✅ **កុំលុប column ដើម** (ល.រ, ទូរស័ព្ទ, លេខកាត, ឈ្មោះ, តួនាទី)

✅ **ប្រើតែនិមិត្តសញ្ញាដែលបានកំណត់** (✓, ○, ✗) ដើម្បីស្ថានភាពត្រឹមត្រូវ

⚠️ **ប្រសិនបើឈ្មោះ ឬលេខកាតមិនត្រូវគ្នា** ប្រព័ន្ធនឹងរំលងកំណត់ត្រានោះ

## បញ្ហាដែលអាចជួបប្រទះ:

### ១. "មិនមានទិន្នន័យសម្រាប់ import"
- ពិនិត្យឯកសារ Excel មានទិន្នន័យឬទេ
- ត្រូវប្រាកដថាមានការកំណត់និមិត្តសញ្ញា (✓, ○, ✗) ក្នុងថ្ងៃណាមួយ

### ២. "Employee not found"
- ពិនិត្យឈ្មោះបុគ្គលិកត្រូវគ្នាជាមួយក្នុងប្រព័ន្ធឬអត់
- ឬពិនិត្យលេខកាតត្រូវគ្នាឬអត់

### ៣. Import បានតែខ្លះ
- ចំពោះកំណត់ត្រាមួយចំនួនដែលបានបរាជ័យ អាចមកពីឈ្មោះ ឬលេខកាតមិនត្រូវគ្នា
- ពិនិត្យ console log ដើម្បីមើលបញ្ហាលម្អិត
