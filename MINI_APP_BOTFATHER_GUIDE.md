# មគ្គុទ្ទេសក៍ការកំណត់ Bot Menu Button ក្នុង Telegram

ដើម្បីឱ្យបុគ្គលិកអាចឃើញប៊ូតុង "បើកកម្មវិធី" (Menu Button) នៅក្បែរកន្លែងសរសេរសារជានិច្ច សូមអនុវត្តតាមជំហានខាងក្រោម៖

### ១. ចូលទៅកាន់ @BotFather
*   បើក Telegram ហើយស្វែងរក **@BotFather**។
*   ចុចប៊ូតុង **Start** ឬវាយបញ្ជា `/mybots`។

### ២. ជ្រើសរើស Bot របស់អ្នក
*   ជ្រើសរើស Bot ដែលអ្នកចង់ប្រើ (ឧទាហរណ៍៖ `@Chantha_Attendance_bot`)។
*   ចុចលើប៊ូតុង **Bot Settings**។
*   ចុចលើប៊ូតុង **Menu Button**។
*   ចុចលើប៊ូតុង **Configure menu button**។

### ៣. បញ្ចូលព័ត៌មាន Mini App
*   **URL**: បញ្ចូលអាសយដ្ឋាន URL សាធារណៈរបស់អ្នក (ឧទាហរណ៍៖ `https://hrms.ksfh.gov.kh/telegram`)។
*   **Title**: ដាក់ឈ្មោះប៊ូតុង (ឧទាហរណ៍៖ `បើកកម្មវិធី` ឬ `KSFH App`)។

### ៤. កំណត់បញ្ជា (Commands) - ជម្រើសបន្ថែម
អ្នកក៏អាចបន្ថែមបញ្ជា (Commands) ដើម្បីឱ្យបុគ្គលិកងាយស្រួលចុច៖
*   ត្រឡប់ទៅ **Bot Settings** > **Edit Bot** > **Edit Commands**។
*   វាយបញ្ចូល៖
    ```text
    start - ចាប់ផ្តើម និងបើកកម្មវិធី
    app - បើកកម្មវិធី Mini App
    help - ជំនួយ និងការប្រើប្រាស់
    ```

---
> [!IMPORTANT]
> - ប៊ូតុង Menu នឹងបង្ហាញសម្រាប់គ្រប់គ្នាដែលប្រើប្រាស់ Bot របស់អ្នក។
> - អ្នកត្រូវប្រើ **HTTPS** URL ដែលអាចចូលប្រើបានជាសាធារណៈ ទើបប៊ូតុងនេះដំណើរការ។
